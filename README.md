# USMCC Paper Publicity Helper

A native macOS app that turns one row pasted from the USMCC paper-submission
Google Sheet into ready-to-post publicity content (Twitter/X, Bluesky,
LinkedIn, plus a plain-language summary) via the Claude API.

Double-click the `.app`, paste a row, click Generate, copy each output.

See [`PLAN.md`](./PLAN.md) for the full implementation plan, input format,
prompt templates, and step-by-step build instructions.
